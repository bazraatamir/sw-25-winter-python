---

##  TetrisGame CLASS

### Үүрэг:
- Тоглоомын ерөнхий удирдлага
- Game loop хянах
- Event, update, draw-г зохицуулах

### Гол функцууд:
- `run()`  
  → Тоглоомын үндсэн давталт  
- `handle_events()`  
  → Keyboard болон window event боловсруулах  
- `update()`  
  → Тоглоомын логик шинэчлэх  
- `draw()`  
  → Дэлгэц дээр бүх зүйлийг зурах  

---

##  Grid CLASS (Тоглоомын талбай)

### Үүрэг:
- Tetris-ийн 2 хэмжээст талбайг удирдах
- Блок байрлуулах
- Мөр дүүрсэн эсэх шалгах

### Гол ойлголт:
- Grid нь row × column бүхий хүснэгт
- Нэг нүд нь нэг блокийг илэрхийлнэ

### Гол функцууд:
- `is_valid_position()`  
  → дүрс зөв байрлалтай эсэх  
- `add_shape()`  
  → дүрсийг тогтмол болгох  
- `clear_full_rows()`  
  → дүүрсэн мөрийг устгах  

---

##  Shape CLASS (Унаж буй дүрс)

### Үүрэг:
- Унаж буй Tetris дүрсийг төлөөлнө
- Хөдөлгөөн, эргэлт хариуцна

### Дүрсний төрлүүд:
- I, O, T, L, S, Z хэлбэрүүд

### Гол функцууд:
- `move_left()` → зүүн тийш шилжих
- `move_right()` → баруун тийш шилжих
- `move_down()` → доош унах
- `rotate()` → 90° эргэх

---

##  Туслах функцууд

### draw_grid()
- Тоглоомын талбайн шугамуудыг зурах

### draw_shape()
- Одоогийн унаж буй дүрсийг дэлгэц дээр зурах

##  Тоглоом хэрхэн ажиллах вэ? (Ерөнхий)

1. Тоглоом эхлэхэд шинэ Shape үүснэ
2. Shape автоматаар доош унана
3. Хэрэглэгч сумтай товчоор удирдана
4. Доошлох боломжгүй үед Grid дээр тогтмол болно
5. Мөр дүүрвэл устгагдана
6. Дээд хэсэг дүүрвэл game over болно

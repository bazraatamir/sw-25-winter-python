# Hangman — Үг таах (Гэрийн даалгаврын заавар)

**Тодорхойлолт:**
Hangman (Дүүжлүүр) бол нэг тоглогч эсвэл хоёр тоглогчтай, нууц үгийг үсэг үгээс бүрдүүлэн таах тоглоом юм. Тоглогчид таасан буруу үсэг бүрт дүүжлүүрийн хэсэг нэмэгдэнэ. Бүх хэсгийг бүрдүүлж амжвал тоглогч хожигдоно; харин үгийг бүрэн таалбал ялна.

---
Програмд ашиглагдах хоёр шинэ ойлголт
1. import гэж юу вэ?

import нь Python-д гаднаас бэлэн бичигдсэн модулийг дуудаж ашиглах боломж олгоно.

Жишээ:

import random

Энэ нь Python-ийн random нэртэй санг дуудаж байна. Ингэснээр санамсаргүй тоо болон санамсаргүй сонголт хийх боломжтой болно.

2. random модуль гэж юу вэ?

random бол санамсаргүй сонголт хийхэд ашиглагддаг Python-ийн стандарт модуль.

Жишээ:

import random
word = random.choice(["PYTHON", "MONGOL", "CODING"])

random.choice(list) нь жагсаалтаас санамсаргүй нэг утгыг сонгоно.

## Даалгаврын шаардлага (заавал биелүүлэх)

1. Програм нь терминал дээр ажиллана (GUI хэрэглэхгүй).
2. Хэрэглэгч нууц үггүү (эсвэл програм санамсаргүйгээр үг сонгож болно).
3. Тоглогч үсэг бүрийг оруулж таана.
4. Таасан зөв үсгүүд нь үгийн тухайн байрлал дээр илрэн харуулна (жишээ: `P _ T H O N`).
5. Таасан буруу үсэг бүрт "дүүжлүүр" (ASCII art) нэг хэсгээр нэмэгдэнэ. Голчлон дараах хэсгүүдийг агуулна: толгой, бие, хоёр гар, хоёр хөл (нийт 6 хэсэг буюу илүү).
6. Хэрэглэгч давтан ижил үсэг оруулбал анхааруулж, дахин нэвтрэх боломж олгоно (боловсорсон validation).
7. Тоглогч үгийг бүрэн таах эсвэл дүүжлүүр бүрдсэн үед тоглолт дуусна.
8. Сайжруулалт (заавал биш — нэмэлт): үг сан (list) -ээс санамсаргүй үг сонгох, түвшин/хялбар/дунд/хүнд, оролтын том жижиг үсгийг үл тоомсорлох.

## ASCII дүүжлүүрийн жишээ (6 шат)

```
  +---+
  |   |
  O   |
 /|\  |
 / \  |
      |
=========
```

Эхлээд хоосон дүүжлүүрээс эхэлж, буруу таах бүрт хэсгүүдийг нэг нэгээр нэмнэ.

---

## Жишээ тоглолтын урсгал

1. Програм: `Нууц үг: _ _ _ _ _`
2. Хэрэглэгч: `A` → Програм: `A _ _ _ _` (зөв бол байрлал харуулна)
3. Хэрэглэгч: `X` → Програм: `Буруу!` → Дүүжлүүрэд толгой нэмэгдэнэ
4. Програм: `Already guessed: X` гэх мэт
